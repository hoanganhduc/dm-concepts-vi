Định nghĩa 1.36. average-case analysis (d.t.) — phân tích trường hợp trung bình.

Phân tích trường hợp trung bình là việc ước lượng chi phí (thời gian chạy hoặc tài nguyên) kỳ vọng của một thuật toán, lấy trung bình trên tất cả các đầu vào có cùng kích thước theo một phân phối xác suất giả định cho các đầu vào (theo [2]).
🔗
Thuật ngữ tiếng Việt tương ứng:
🔗
  • phân tích trường hợp trung bình [24] — cách dùng phổ biến.
    🔗
    🔗
  • phân tích trung bình (biến thể, chưa có trích dẫn cụ thể).
    🔗
    🔗
Ví dụ: Với phép tìm kiếm tuyến tính trên \(n\) phần tử, nếu phần tử cần tìm xuất hiện ở mỗi vị trí với xác suất như nhau thì số phép so sánh trung bình xấp xỉ \(\displaystyle\frac{n+1}{2}.\)
🔗
Xem thêm asymptotic analysis (d.t.) — phân tích tiệm cận, best-case analysis (d.t.) — phân tích trường hợp tốt nhất.
