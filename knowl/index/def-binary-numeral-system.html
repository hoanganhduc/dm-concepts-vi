Định nghĩa 2.15. binary numeral system (d.t.) — hệ nhị phân.

Hệ nhị phân (hệ cơ số \(2\)) là hệ đếm dùng cơ số \(b = 2\) với hai chữ số \(0\) và \(1\text{.}\) Mỗi số nguyên không âm \(n\) có duy nhất một khai triển nhị phân \(n = a_{k-1} 2^{k-1} + \dots + a_1 2^1 + a_0 2^0\text{,}\) trong đó mỗi chữ số \(a_i \in \{0, 1\}\text{,}\) ký hiệu \((a_{k-1} \dots a_1 a_0)_2\) (theo [1], Rosen §4.2).
