Định nghĩa 3.7. ceiling function (d.t.) — hàm trần.

Hàm trần là hàm gán cho mỗi số thực \(x\) số nguyên nhỏ nhất lớn hơn hoặc bằng \(x\text{,}\) ký hiệu \(\lceil x \rceil\text{.}\) Như vậy \(\lceil x \rceil\) là số nguyên \(n\) duy nhất thỏa mãn \(n - 1 \lt x \leq n\) (theo [1], Rosen §2.3).
🔗

    🔗
Ví dụ: Chẳng hạn \(\lceil 3.1 \rceil = 4\text{,}\) \(\lceil -1.5 \rceil = -1\) và \(\lceil 5 \rceil = 5\text{.}\)
