Định nghĩa 7.21. greatest integer function (d.t.) — hàm phần nguyên.

Hàm phần nguyên (còn gọi là hàm sàn, ký hiệu \(\lfloor x \rfloor\)) là hàm gán cho mỗi số thực \(x\) số nguyên lớn nhất không vượt quá \(x\text{,}\) tức là số nguyên lớn nhất có giá trị \(\leq x\) (theo [1], Rosen §2.3).
🔗
Thuật ngữ tiếng Việt tương ứng:
🔗
  • hàm phần nguyên [11] (tr. 86) — cách dùng phổ biến.
    🔗
    🔗
  • hàm sàn [25] (tr. 171); [11] (tr. 86); [23] (tr. 52); [29].
    🔗
    🔗
  • phần nguyên [22] (tr. 91); [11] (tr. 86); [27] (tr. 80); [29]; [28] (tr. 64).
    🔗
    🔗
Ví dụ: Ta có \(\lfloor 2.7 \rfloor = 2\text{,}\) \(\lfloor 3 \rfloor = 3\) và \(\lfloor -1.5 \rfloor = -2\text{.}\)
