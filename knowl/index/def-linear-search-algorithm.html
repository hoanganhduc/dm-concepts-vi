Định nghĩa 12.36. linear search algorithm (d.t.) — thuật toán tìm kiếm tuyến tính.

Thuật toán tìm kiếm tuyến tính là một thủ tục để tìm kiếm một phần tử trong một bảng kê bằng cách duyệt lần lượt từng phần tử theo thứ tự. Thuật toán so sánh phần tử cần tìm với từng phần tử của bảng kê, theo đúng thứ tự xuất hiện, cho tới khi tìm thấy phần tử đó hoặc duyệt hết bảng kê (theo [1], Rosen §3.1).

Ví dụ: Để tìm \(x = 9\) trong bảng kê \(1, 3, 5, 7, 9, 11\text{,}\) ta lần lượt so sánh \(x\) với \(1\text{,}\) rồi \(3\text{,}\) rồi \(5\text{,}\) rồi \(7\text{,}\) rồi \(9\) và dừng lại khi gặp \(9\) ở vị trí thứ năm; trong trường hợp xấu nhất thuật toán cần \(n\) phép so sánh, tức độ phức tạp \(O(n)\text{.}\)
