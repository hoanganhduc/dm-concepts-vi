Định nghĩa 17.5. quotient set (d.t.) — tập hợp thương.

Tập hợp thương của một tập hợp \(A\) theo một quan hệ tương đương \(R\) trên \(A\text{,}\) ký hiệu \(A/R\text{,}\) là tập hợp gồm tất cả các lớp tương đương của các phần tử của \(A\) theo \(R\text{,}\) tức là \(A/R = \{[a]_R : a \in A\}\text{.}\) Các lớp tương đương này tạo thành một phân hoạch của \(A\) (theo [1], Rosen §9.5).

Ví dụ: Xét tập hợp \(A = \{0, 1, 2, 3, 4, 5\}\) với quan hệ tương đương \(R\) xác định bởi \(a \mathrel{R} b\) khi và chỉ khi \(a \equiv b \pmod{3}\text{.}\) Khi đó các lớp tương đương là \([0]_R = \{0, 3\}\text{,}\) \([1]_R = \{1, 4\}\) và \([2]_R = \{2, 5\}\text{,}\) nên tập hợp thương là \(A/R = \{\{0, 3\}, \{1, 4\}, \{2, 5\}\}\text{.}\)
