Định nghĩa 19.14. set difference (d.t.) — hiệu của hai tập hợp.

Hiệu của hai tập hợp \(A\) và \(B\text{,}\) ký hiệu \(A - B\) (hoặc \(A \setminus B\)), là tập hợp gồm các phần tử thuộc \(A\) nhưng không thuộc \(B\text{;}\) nghĩa là \(A - B = \{x : x \in A \wedge x \notin B\}\) (theo [1], Rosen §2.2).

Ví dụ: Với \(A = \{1, 2, 3, 5\}\) và \(B = \{2, 4, 5\}\) thì \(A - B = \{1, 3\}\) và \(B - A = \{4\}\text{.}\) Nói chung \(A - B \neq B - A\text{.}\)
