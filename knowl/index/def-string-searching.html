Định nghĩa 19.42. string searching (d.t.) — so khớp chuỗi.

So khớp chuỗi là bài toán xác định mọi vị trí xuất hiện của một chuỗi mẫu trong một chuỗi văn bản (theo [1], Rosen §3.3).
🔗

    🔗

    🔗

    🔗

Ví dụ: Trong chuỗi văn bản \(T = \texttt{abracadabra}\text{,}\) việc so khớp chuỗi mẫu \(P = \texttt{abra}\) trả về các vị trí xuất hiện bắt đầu tại chỉ số \(0\) và \(7\text{.}\)
