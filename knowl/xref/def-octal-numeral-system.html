Định nghĩa 15.3. octal numeral system (d.t.) — hệ bát phân.

Hệ bát phân (hệ cơ số \(8\)) là hệ đếm dùng cơ số \(b = 8\) với tập chữ số \(\{0,1,2,3,4,5,6,7\}\text{.}\) Mỗi số nguyên không âm \(n\) được biểu diễn duy nhất dưới dạng khai triển cơ số \(8\text{,}\) ký hiệu \((n)_8\text{,}\) theo công thức \(n = a_k 8^k + a_{k-1} 8^{k-1} + \cdots + a_1 8 + a_0\text{,}\) trong đó \(0 \leq a_i \leq 7\) (theo [1], Rosen §4.2).
