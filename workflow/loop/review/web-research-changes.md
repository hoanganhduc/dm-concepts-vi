# Đề xuất đổi thuật ngữ từ nghiên cứu web (chờ duyệt) — 6/7 lô

## Đồ thị (pilot)
- open walk: không khép kín → **đường đi mở** (Wikipedia, đã xác minh)
- nonseparable graph: không thể tách rời → **đồ thị song liên thông**
- subdivision: phép phân chia cạnh → **phép chia cạnh**
- descendants: đỉnh con cháu → **hậu duệ** (cần xác minh thêm)
- left/right child: đỉnh con trái/phải → **con trái/con phải**
- no_source: Johnson graph, Kuratowski's subgraph

## A. Số học & mật mã
- one's complement notation → **biểu diễn bù 1**
- two's complement notation → **biểu diễn bù hai**
- shift cipher: mật mã dịch → **mã dịch vòng**
- cryptosystem: hệ thống mật mã → **hệ mật mã**
- block cipher → mã hóa khối (Wikipedia) [hoặc giữ mật mã khối]
- no_source: character cipher

## B. Logic
- circular reasoning: lý luận quẩn → **lập luận vòng vo**
- existential generalization: hiện sinh → **tổng quát hóa tồn tại**
- existential instantiation: khởi tạo hiện sinh → **đặc biệt hóa tồn tại**
- universal instantiation: khởi tạo phổ quát → **cụ thể hóa phổ dụng**
- fallacy of affirming the conclusion: hậu kiện → **khẳng định kết luận**

## C. Tập hợp/hàm/dãy
- codomain: miền giá trị → **tập đích** (tránh nhầm range)
- geometric series: tổng cấp số nhân → **chuỗi hình học**
- rational numbers: hữu tỷ → **hữu tỉ**
- strictly decreasing: thực sự giảm → **giảm nghiêm ngặt**
- set builder notation → chỉ ra tính chất đặc trưng (SGK)
- membership table: bảng tính thuộc → bảng thành viên (độ chắc TB)

## D. Thuật toán & automata
- nondeterministic finite automaton: không tất định → **ôtômat hữu hạn không đơn định**
- nondeterministic Turing machine: không tất định → **máy Turing không đơn định**
- stable matching: bổ sung biến thể "bền vững"

## E. Data-mining/khác + Đếm
- extremal graph theory: cực hạn → **cực trị** (QUAN TRỌNG, Wikipedia)
- selection operator: toán tử chọn → **phép chọn (σ)**

## G. Boole/mạch + Xác suất
- mutually independent events: độc lập tương hỗ → **độc lập toàn phần**

## F. Cây + Quan hệ + Quy nạp
- minmax strategy: minmax → **minimax** (Wikipedia)
- root: đỉnh gốc → **nút gốc** (nếu theo CTDL; "đỉnh gốc" ổn nếu theo đồ thị)
- no_source: universal address system
